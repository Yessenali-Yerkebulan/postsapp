## Blog App

Commands to run project

```sh

git clone https://github.com/Yessenali-Yerkebulan/postsapp.git

```

```sh

composer install

```

```sh

cp .env.example .env

```

```sh

php artisan migrate

```

```sh

php artisan serve

```

```sh

npm run dev

```

Commands to work  with queues and scheduler

```sh

php artisan queue:work

```

```sh

php artisan schedule:work

```
